Kompilieren Sie mittels des Source Codes aus dem Ordner mysql-8.0.12_source MySQL Software.
Die Installation soll in den Pfad /opt/mysql/mysql-8-0.12 erfolgen.

Nutzen Sie die folgenden Kommandos für die Lösung dieser Aufgabe:

- ccmake .
- make
- make install

Bitte nehmen Sie derzeit KEINE (!!!) Änderung am Link "CURRENT" vor.
